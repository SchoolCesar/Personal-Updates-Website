# Personal Updates Website

## Overview

This is a personal website that I built to document what I am currently working on and where I am heading professionally.

The main idea behind this project is simple — instead of letting my progress stay scattered across notes or ideas, I wanted one place where I can consistently update what I’m learning, building, and improving over time.

Because of that, this site is meant to evolve. It’s not a finished product, but something I will continue to come back to and update.

---

## Purpose

I built this project for two main reasons:

- To practice and improve my frontend development skills (HTML, CSS, JavaScript)
- To create a space where I can track and reflect on my progress over time

I think having something like this makes my learning more intentional, because I’m not just building projects — I’m also documenting the process behind them.

---

## Features

- Clean and simple layout
- Sections for current goals and updates
- Easy to update content over time
- Deployed and accessible online
